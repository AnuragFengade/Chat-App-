# Chat-App-
💬 Chat Application

A real-time chat application that allows users to communicate instantly with a clean and responsive interface. This project demonstrates modern web development practices including real-time messaging, user authentication, and scalable backend architecture.

🚀 Features
🔐 User Authentication (Login / Signup)
💬 Real-time Messaging
🟢 Online/Offline Status
📩 Private & Group Chats
📎 File/Image Sharing (optional)
🔔 Notifications
📱 Responsive UI (Mobile + Desktop)
🛠️ Tech Stack

Frontend:

HTML, CSS, JavaScript / React (if used)

Backend:

Java / Spring Boot / Node.js

Database:

MySQL / MongoDB

Other Tools:

WebSockets / Socket.IO
REST APIs

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/chat-app.git
cd chat-app
2️⃣ Backend Setup
cd backend
mvn spring-boot:run

(or for Node.js)

npm install
npm start
3️⃣ Frontend Setup
cd frontend
npm install
npm start
🔑 Environment Variables

Create a .env file and add:

PORT=8080
DB_URL=your_database_url
JWT_SECRET=your_secret_key
